This is a minimal Angular 20 standalone-component demo app for a 1-hour Playwright + AI test generation workshop.

Features:
- Angular Demo App (Policy Manager)
- Playwright example tests
- Prompt files for AI-assisted test generation
- Best-practices and workshop schedule

How to run (attendees):
```bash
# install deps
npm install

# install playwright browsers
npx playwright install

# start dev server
npm run start

# in another terminal, run tests
npx playwright test
```

Project structure:
- 01-angular-app/: Angular app
- 02-prompts/: prompts and dummy feature files
- 03-best-practices/: Page objects and guidelines
- WORKSHOP.md: 1-hour schedule
