# Demo Script (Instructor)

1. cd demo-app
2. npm install
3. npx playwright install
5. npx playwright init-agents --loop=vscode
4. npm run start
6. In VS Code Copilot Chat: run @planner with example-planner-prompts.md
7. Inspect generated specs/ files
8. Run @generator on a spec and inspect tests/
9. Run npx playwright test
10. If failures, run @healer to repair
