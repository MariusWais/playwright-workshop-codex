# Getting Started
## Prerequisites

- **Node.js 18+** - [Download](https://nodejs.org/)
- **VS Code v1.105+** - [Download](https://code.visualstudio.com/)
- **Playwright Test for VS Code** extension - [Install](vscode:extension/ms-playwright.playwright) (or search in VS Code Extensions)
- **GitHub Copilot** - [Subscribe](https://github.com/features/copilot/plans)

## Setup

```bash
git clone https://github.com/MariusWais/playwright-workshop-codex.git
cd playwright-workshop-codex
npm install && npx playwright install
npm start              # Terminal 1 - keep running
```

Open http://localhost:4222

## Run Tests (1 minute)

```bash
npx playwright test --headed   # Terminal 2
```

Report opens automatically when done.
